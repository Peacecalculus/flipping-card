## How to Customize ğŸ’¡

- Change the images: replace `images/*` with your files and update the `src` in `card.html`.
- Card size: edit the dimensions in `htmlcss/style.css` (look for width/height variables or the `.card` rules).
- Flip direction: modify transform rules in `htmlcss/style.css` (rotateX / rotateY) to change the flip axis.
- Styling: adjust background colors, fonts, and shadow to match your design system.
